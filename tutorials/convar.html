Getting Started

Okay, now i'm gonna explain how to make a ConVar by lua.

The first one, go to the ../lua/autorun/ and create here random .lua script, it doesn't matter what you call it.

Doing it!

Basic function for creation ConVar is:

CreateConVar( "name", "defval", flag, "helptext", "minval", "maxval" )

In fact, it is not important to specify all the arguments, the first 3 are enough for a minimum.

Here are some example of ConVars in use:

 
local checkBool = CreateConVar("gmed_giveweapons", "1", FCVAR_CLIENT, "0", "1")

if (checkBool:GetBool() == true) then
	print("cvar is working")
end

--========

local checkInt = CreateConVar("checkInteger", "1", FCVAR_SERVER, "0", "255")

if (checkInt:GetInt() == 133) then
	print("Answer is 133!")
end


ConVar Flags:

 
FCVAR_CLIENT
FCVAR_SERVER
FCVAR_NONE
