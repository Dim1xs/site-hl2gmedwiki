Creating a Nextbot

Description:How to make a nextbot.
Author:Dim1xs
Created:April 13th 2024

The Beggining

This tutorial will show you how to make a simple Nextbot, with using Nextbot Base.

Make sure you have installed Nextbot Base. (it's on discord server)

Look here for Nextbot Hook list

Starting from script

You can make a new script, or just copy a example, it is in the srceng/hl2/custom/nextbots base/lua/entities/nextbot_example

Open ðŸ”µshared.lua 
      
ENT.Name = "[NEXTBOT] Example" // Name in SpawnMenu
ENT.Base = "nextbot_base"
                  
//Settings:
ENT.Texture = "nextbot_base/nextbot_base.vmt" // Path to texture. [ONLY .VMT!]
ENT.Scale = 0.5 // Scale of Texture. [ BETTER DONT TOUCH! ]
ENT.Distance = 16192 // Distance to find players/doors/props.
ENT.Speed = 2.5 // Speed of nextbot movement.
ENT.JumpCooldown = 1.5 // How many seconds will it be possible to jump again.
                  
ENT.IdleSound = "base/idle.wav" // Idle sound, plays everytime [path]
ENT.IdleDuration = 2 // in Seconds [ Required for auto sound refresh. ]
ENT.IdlePlayEverywhere = false // Play Idle Sound Everywhere?
ENT.IdlePitch = 100 // Pitch.
ENT.IdleVolume = 1  // Volume.
ENT.IdleDistance = 75 // Distance Multiplier. {MIN: 55}
                  
ENT.KillWalkSound = "base/killwalk.wav" // What sound should play on killing walking player? [path]
ENT.KillRunSound = "base/killrun.wav" // What sound should play on killing running player? [path]
ENT.JumpSound = "base/jump.wav" // What sound should play on nextbot jump? [path]
ENT.Murder = true // if false then it don't kills anyone.
ENT.Damage = 25 // WIP.
ENT.AttackOnlyPlayer = true // Dont touches NPC if true. [ Kinda broken ]
ENT.ThrowRagdolls = true //Throw ragdolls on touch.
                  
//Props
ENT.UnfreezeProps = true // Will nextbot unfreeze props that it touches?
                  
//Doors settings
ENT.BreakDoors = true // Will nextbot break doors that it touches?
ENT.BreakDoorSound = "physics/wood/wood_plank_break1.wav" // What sound should play on door break? [path]
ENT.BreakDoorType = 1 // Flag for door breaking
// 1 = Break and throw door
// 2 = Remove the door
ENT.BreakDoorCooldown = 1.5 // in Seconds
                  
//Searching [BROKEN]
ENT.SearchForProps = false // This thing can broke nextbot and it will just stay. [ FIXME ]
ENT.SearchForDoors = false // This thing can broke nextbot and it will just stay. [ FIXME ]

//OTHER
ENT.PlayerRunSpeed = 321 // for detecting [ BETTER DONT TOUCH! ]
ENT.PlayerWalkSpeed = 191 // for detecting  [ BETTER DONT TOUCH! ] 
ENT.PlayerNoclipSpeed = 1000 // for detecting (better more than 1000) [ BETTER DONT TOUCH! ]
                  
//TODO:
ENT.CollisionSize = 1
Make sure that your Nextbot texture is in VTF format, and path is set to VMT!
