Nextbot hooks

This tutorial will show you how to make a simple Nextbot, with using Nextbot Base.

Name Description
NextbotOnJump
Called when nextbot jumps.
NextbotOnError
Called when nextbot got error on initialization.
NextbotOnDump
Called when nextbot does dump.
NextbotOnDebugToggle
Called when nextbot debugging toggled.
NextbotOnDamage
Called when nextbot kill someone.
NextbotOnDoorBreak
Called when nextbot brokes the door.
NextbotOnRagdollThrow
Called when nextbot throws the ragdoll.
NextbotOnRotate
[UNUSED] Test function for SCP173, called when it rotates to player direction.
NextbotTextureInitialize
Called after GlobalInit and ModelInit.
NextbotInitialize
Called after all initializations are done.
NextbotGlobalInitialize
Called by the very first init.
NextbotModelSetup
Same as TextureInit, but called after GlobalInit and before Initialization.
NextbotThink
Called every frame.
