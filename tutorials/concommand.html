Description: Guide to create scripted Command.
Author: Dim1xs

Getting Started

Okay, this tutorial gonna explain how to make a custom lua command.

The first one, go to the ../lua/autorun/ path and create here random .lua script, it doesn't matter what you call it.

Doing it!

Basic function for creation ConCommand is:

concommand.Add( "commandname", functionname )

In fact, you can create function into it.

Here are some example of ConCommand in use:

 
local function RunPrint()
	print("print is done")
end

concommand.Add("runPrint", RunPrint)

--========

concommand.Add("printRun", function()
	print("print is done")
end)


You also can remove it!

Basic function for creation ConCommand is:

concommand.Remove( "commandname" )

Example:

 
concommand.Remove("runPrint")

--========

concommand.Remove("printRun")


That's all!
