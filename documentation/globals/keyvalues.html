KeyValues

KeyValues KeyValues( string name, string firstKey, any firstValue)

Arguments:

  • 1) string name

    • Name for KeyValue.

  • 2) string firstKey

    • First key for KeyValue.

  • 3) any firstValue

    • First Value for KeyValue. Can be int or string.

Description:

Creates and returns KeyValues object.

Returns:

  • 1) KeyValues

    • The KeyValues object.
