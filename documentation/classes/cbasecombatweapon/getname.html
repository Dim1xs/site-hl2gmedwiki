GetName

Returns the map/hammer targetname of this entity. 

Weapon:GetName()

Returns:

  • 1) string Name

    • Name on map.

Example:


function getName( pPlayer )
   print( pPlayer:GetActiveWeapon():GetName() ) --Returns entity name of da weapon. 
end

concommand.Add("getName", getName)
