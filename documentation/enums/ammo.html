AMMO

Enumerations used by game.AddAmmoType.

Values:


ENUM NUM Description
AMMO_FORCE_DROP_IF_CARRIED 1 Forces player to drop the object they are carrying if the object was hit by this ammo type.
AMMO_INTERPRET_PLRDAMAGE_AS_DAMAGE_TO_PLAYER 2 Uses plydmg of the ammo type as the damage to deal to shot players instead of bullet damage.
