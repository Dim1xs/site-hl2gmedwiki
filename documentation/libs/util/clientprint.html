UTIL.ClientPrint

Displays a message either in their chat, console, or center of the screen. 

UTIL.ClientPrint( Entity player , number type , string message  )

Arguments:

  • 1) Entity player

    • Entity of player to be receiver of message.

  • 2) number type

    • You can find a list of print types here.

  • 3) string message

    • Message to be sent to the player.

Example:

function clientPrint( pPlayer )
   UTIL.ClientPrint( pPlayer, 1, "Test Message" )
   UTIL.ClientPrint( pPlayer, 2, "Test Message" )
   UTIL.ClientPrint( pPlayer, 3, "Test Message" )
   UTIL.ClientPrint( pPlayer, 4, "Test Message" )
end
