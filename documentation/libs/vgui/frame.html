vgui.Frame

Creates vgui button. 

Frame vgui.Frame( Panel parent, string panelName, bool showTaskbarIcon, bool bPopup )

Arguments:

  • 1) Panel parent

    • Parent of the button.

  • 2) string panelName

    • Name of the button.

  • 3) bool showTaskbarIcon

    • Show taskbar icon.

  • 4) bool bPopup

    • Should popup on create?

Returns:

  • 1) Frame
