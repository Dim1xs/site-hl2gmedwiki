HL2GMED - Half-Life 2 Modification


HL2GMED - Half-Life 2 Garry's Mod Edition.

HL2GMED is a Half-Life 2 Modification, created on a modified Source Engine from Nillerusr.
This modification allows you to immerse yourself in the world of the physical abilities of the Source engine.

The most important thing is that the Lua scripting language is present in the modification, it allows players to add new elements for a more interesting game process.

The modification aims to allow the player to get the same playing conditions as in Garry's mod, but only on a mobile device.
